## 📁 sql-ex_ru

В этой папке собраны решения задач с [sql-ex.ru](https://www.sql-ex.ru).  
Затрагиваемые темы:

- работа с несколькими таблицами (`Product`, `PC`, `Laptop`, `Printer`);
- агрегация, подзапросы, оконные функции;
- даты, интервалы, расширение диапазонов;
- нумерация строк по разным правилам.

**Задачи:** 25, 55, 75, 105, 150.

## 📁 StreamFlex

Здесь моделируется аналитики для стримингового видеосервиса.  
Схема `sf` включает таблицы:

- `users` – пользователи, реферальные связи;
- `plans` – тарифные планы;
- `subscriptions` – подписки;
- `payments` – платежи и возвраты;
- `shows` – каталог контента;
- `viewings` – события просмотров.

**9 задач** на темы:

1. **Хиты по странам** – фильтрация групп, оконные функции.
2. **Платные vs бесплатные пользователи** – EXISTS / NOT EXISTS.
3. **Средний платёж по странам** – агрегация через цепочку таблиц.
4. **Чистая и грязная выручка** – CTE, UNION, агрегация.
5. **Топ-2 устройства для каждого пользователя** – оконные функции, ранжирование.
6. **Скользящее 7-дневное окно** – `ROWS BETWEEN`, генерация календаря.
7. **Дерево рефералов** – рекурсивный CTE.
8. **CTE с повторным использованием** – один CTE, две разных выборки через `UNION ALL`.
9. **Изменяемое представление с CHECK OPTION** – DDL, вставка с проверкой.
